# 🏥 병원 환자 전광판 프로토타입

React + Vite + Tailwind CSS로 구현된 병원 환자 대기 전광판 시스템입니다.

## ✨ 주요 기능

- **실시간 환자 대기 현황 표시**: 환자 번호, 이름, 진료과, 담당의사 정보
- **동적 상태 업데이트**: 대기중, 호출중, 완료 상태를 실시간으로 표시
- **의료진 현황**: 각 의사의 현재 상태 (진료 가능, 진료중, 휴식중)
- **실시간 시계**: 현재 시간과 날짜 표시
- **대기 통계**: 현재 대기 환자 수와 호출 환자 수
- **공지사항**: 병원 공지사항 표시
- **반응형 디자인**: 다양한 화면 크기에 최적화

## 🚀 시작하기

### 필수 요구사항

- Node.js (v16 이상)
- npm 또는 yarn

### 설치 및 실행

1. 의존성 설치:
```bash
npm install
```

2. 개발 서버 실행:
```bash
npm run dev
```

3. 브라우저에서 `http://localhost:5173` 접속

### 빌드

```bash
npm run build
```

빌드된 파일은 `dist` 폴더에 생성됩니다.

## 🏗️ 프로젝트 구조

```
src/
├── components/
│   ├── HospitalBoard.jsx     # 메인 전광판 컴포넌트
│   ├── PatientQueue.jsx      # 환자 대기 목록
│   ├── CurrentTime.jsx       # 실시간 시계
│   └── DoctorStatus.jsx      # 의료진 현황
├── App.jsx                   # 메인 앱 컴포넌트
├── App.css                   # 커스텀 스타일
├── index.css                 # 글로벌 스타일 (Tailwind)
└── main.jsx                  # 앱 진입점
```

## 🎨 디자인 특징

- **모던한 글래스모피즘 디자인**: 투명한 배경과 블러 효과
- **다크 테마**: 전광판에 적합한 어두운 배경
- **애니메이션**: 호출중인 환자는 펄스 애니메이션으로 강조
- **한글 폰트**: Noto Sans KR로 한글 가독성 최적화
- **디지털 폰트**: 환자 번호는 디지털 시계 스타일 폰트 사용

## 📱 반응형 디자인

- **데스크톱**: 3열 레이아웃 (환자 목록 + 의료진 현황 + 공지사항)
- **태블릿/모바일**: 1열 스택 레이아웃으로 자동 조정

## 🔧 기술 스택

- **React 18**: 사용자 인터페이스 구축
- **Vite**: 빠른 개발 서버와 빌드 도구
- **Tailwind CSS**: 유틸리티 우선 CSS 프레임워크
- **Lucide React**: 아이콘 라이브러리

## 📋 시뮬레이션 기능

- 대기 시간이 1분마다 자동으로 증가
- 10초마다 랜덤하게 환자 상태가 변경될 수 있음
- 실제 병원 시스템과 연동하여 실시간 데이터를 받을 수 있도록 구조 설계
